## Compliance & Data Privacy

- This project is a **technical demo** for healthcare integration concepts
  (HL7 v2, FHIR R4, and X12).
- All data is **synthetic** and does **not** represent real patients
  or real claims.
- The demo is **not** a production-ready, HIPAA-compliant system.
- No PHI (Protected Health Information) should be sent to the public demo
  deployment.
